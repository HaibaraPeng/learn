# OpenCode 项目地图

## 项目概览

**项目名称**: OpenCode
**版本**: 1.3.2
**类型**: AI 驱动的编程助手工具
**语言**: TypeScript
**运行时**: Bun
**许可证**: MIT

### 一句话描述

开源 AI 编程助手，类似 Claude Code，支持多模型提供商，内置 LSP 和 MCP 支持。

## 核心包详细结构

### packages/opencode (核心)

**职责**: CLI 入口、AI Agent 逻辑、会话管理、工具系统

```
src/
├── agent/           # Agent 定义和生成
│   ├── agent.ts     # Agent 接口和实现
│   └── prompt/      # Agent 提示词模板
├── session/         # 会话管理
│   ├── index.ts     # 会话 CRUD
│   ├── message.ts   # 消息处理
│   ├── llm.ts       # LLM 交互
│   └── prompt/      # 会话提示词
├── provider/        # AI 模型提供商
│   ├── provider.ts  # 提供商统一接口
│   ├── models.ts    # 模型定义
│   └── transform.ts # 响应转换
├── tool/            # 工具系统
│   ├── bash.ts      # Bash 执行
│   ├── file.ts      # 文件操作
│   ├── edit.ts      # 文件编辑
│   └── ...
├── mcp/             # MCP (Model Context Protocol) 集成
│   ├── index.ts     # MCP 客户端
│   └── auth.ts      # MCP 认证
├── lsp/             # LSP 集成
│   ├── client.ts    # LSP 客户端
│   └── server.ts    # LSP 服务配置
├── cli/             # CLI 命令
│   ├── cmd/         # 各种命令实现
│   └── ui.ts        # TUI 界面
├── project/         # 项目管理
├── storage/         # 数据存储 (SQLite)
├── permission/      # 权限系统
├── bus/             # 事件总线
├── config/          # 配置管理
├── auth/            # 认证系统
├── skill/           # 技能系统
├── effect/          # Effect 服务
└── util/            # 工具函数
```

## 入口文件与核心调用链

### CLI 入口

```
packages/opencode/src/index.ts
    ↓
CLI 命令解析 (yargs)
    ↓
各命令处理器 (cli/cmd/*.ts)
    ↓
RunCommand → TUI 启动
    ↓
Session 创建 → Agent 执行
    ↓
Tool 调用 → Provider API
```

### Web 入口

```
packages/app/src/app.tsx
    ↓
SDK 连接 (context/sdk.tsx)
    ↓
会话列表 (pages/session.tsx)
    ↓
消息交互 (components/prompt-input.tsx)
```
